== pp, profile position mode	操纵位置 模式
     => gi
	 
		    a target position
			  -> trajectory generator -> (position demand value) 
	  		 		  	 			轨迹生成器
			  -> Position control function -> control effort
				    位置控制环

		    Trajectory Generator 更详细的部分
			position	               |  
			velocity     -> Multiplier |  Trajectory Generator -> ( position demand value )
			acceleration -> Multiplier |
		   
     => od entries
	 	  0x607A target position			;目标位置（ 可以是绝对值 或者 是相对值 )
		  		  应该移动到的位置
				  使用用户定义的位置单位给出这个数据
				  
		  0x607B position range limit		;位置范围限制
		  		  min position range limit
				  max
				  限制了输入值的范围
  		  0x607D software position limit	;软件位置限制
		  		  min position range limit
				  max
  		  0x607F max profile velocity		;最大 操纵速度
		  		  最大允许速度
		  0x6080 max motor speed			; 电机最大速度
		  		  电机的最大允许速度(单位是 rpm )
		  0x6081 Profile velocity			; 操纵速度
		  		  
		  0x6082 End velocity				; 末端速度
		  		   drive 到达目标位置时，应该具有有速度
				   ( the drive must have on reaching the target position )
		  0x6083 profile acceleration	; 操纵加速度
		  		   使用用户定义的单位
				   被转換为 increments/second^2
		  0x6084 profile deceleration	; 操纵减速
		  		 		 
		  0x6085 Quick stop deceleration  ;
		  		   用来 停止电机的。
		  0x6086 Motion profile type	  ; 运动操纵类型
		  		   用来 选择 操纵运动的类型
  		  0x60C5 Max acceleration		  ; 最大加速度
		  		   限制加速
		  0x60C6 Max deceleration		  ; 最大减速度
		  		   限制减速
	 => func desc
	      两种方式，让驱动器 应用 target positions
		      Set of set-points,  到达一个目标点，接着执行下一个目标点。（速度不减速到0)
			  Single set-point,   到达一个目标点， 发状态给 host, 然后 接收新的 set-point.
			  		 			  (到目标点后，一般速度都会减为0，再移动到下一个 set-point)

			  相关字段:
				  bit 'change set immediately'	(controlword  bit 5)
				       相关字段	  bit 4				bit 5
				  	   pp mode	  New set-point 	Change set immediately
					   	  		  	  				0	   
													1
				  => bit 'change set immediately' 为 0时， （单个 set-point 模式) ;; 
					     主机准备完数据，发送数据有效的信号。
							 (在controlword把bit4, 即'new set-point'置1)
					   	 驱动器识别并且存储新的有效数据后，回应
						     (在statuword里把bit12, 即'set-point acknowledge'置1)
							 置1表示：	Trajectory generator has assumed the positioning values.
							 
 					     主机收到结果后，知道驱动器已经在处理了，所以可以把 bit4置0了。
						 
					   	 之后还可能收到 驱动器把 bit12置0，
						 	 表示驱动器又能处理新的数据了。	 				
				  => bit 'change set immediately' 为 1时， （ 连续 模式 )
					     连续运行，速度不会降到0.
						 
g	 ; general
gi   ; general information
desc ; description
obj  ; object
func ; functional
fun  ; function
